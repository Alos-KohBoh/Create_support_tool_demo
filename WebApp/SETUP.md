# セットアップ手順

## 初回起動時の注意

初回起動時、または更新後にデータが古い場合は、以下の手順でリセットしてください。

### 方法1: データリセットボタンを使用

1. `index.html` をブラウザで開く
2. 右側の「データ管理」カードにある「🔄 データリセット」ボタンをクリック
3. 確認ダイアログで「OK」をクリック
4. ページが自動的にリロードされ、最新のサンプルデータが読み込まれます

### 方法2: ブラウザのLocalStorageをクリア

#### Chrome / Edge
1. F12キーを押して開発者ツールを開く
2. 「Application」タブをクリック
3. 左側メニューの「Local Storage」を展開
4. サイトのURLをクリック
5. `rpg_monsters` と `rpg_items` を右クリックして「Delete」
6. ページをリロード (F5)

#### Firefox
1. F12キーを押して開発者ツールを開く
2. 「ストレージ」タブをクリック
3. 左側メニューの「ローカルストレージ」を展開
4. サイトのURLをクリック
5. `rpg_monsters` と `rpg_items` を右クリックして「項目を削除」
6. ページをリロード (F5)

### 方法3: シークレット/プライベートモードで開く

- **Chrome**: Ctrl + Shift + N
- **Firefox**: Ctrl + Shift + P
- **Edge**: Ctrl + Shift + N

シークレットモードではLocalStorageが使用されないため、常に最新のサンプルデータが読み込まれます。

## サンプルデータの確認

正しく読み込まれると、以下のモンスターが表示されます：
- スライム
- レッサーウルフ
- ホーンラビット

以下のアイテムが含まれます：
- スライムゼリー
- レッサーウルフの毛
- 角ウサギの毛皮
- など15種類

## トラブルシューティング

### モンスターが「ゴブリン」「ドラゴン」と表示される
→ 古いデータが残っています。上記のリセット方法を実行してください。

### アイテム種類が「回復」「防具」と表示される
→ 古いデータが残っています。上記のリセット方法を実行してください。

### 変更が反映されない
→ ブラウザのキャッシュをクリアするか、Ctrl + F5 でスーパーリロードしてください。
